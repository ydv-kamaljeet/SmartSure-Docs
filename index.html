SmartSure 🚀

Microservices-based Insurance Platform with Event-Driven Architecture

System Overview

SmartSure is a microservices-based insurance platform with Identity, Policy, Claims, Admin, and AI services communicating via RabbitMQ.

Tech Stack

  • Angular 19
  • .NET Core 10
  • RabbitMQ + MassTransit
  • SQL Server
  • Ollama AI

High Level Architecture

All requests pass through Ocelot API Gateway and are routed to services. Each service owns its database and communicates via events.

Low Level Design (Claims & Policy)

Detailed interaction between services, external integrations (Razorpay, SMTP, MEGA storage), and internal APIs.

Event Driven Architecture

  • UserRegisteredEvent
  • PolicyCreatedEvent
  • ClaimSubmittedEvent
  • ClaimApprovedEvent

Use Cases

  • Guest: Register/Login
  • User: Buy Policy, Claims
  • Admin: Manage system
